# Reto Forense: Memorias Perdidas

**Autor**: InspectorForensic

## Descripción
Durante una investigación de respuesta a incidentes en una estación de trabajo Windows, obtuvimos un volcado de memoria RAM sospechoso. Sospechamos que el usuario ejecutó un troyano de acceso remoto (RAT) y exfiltró información sensible.

Tu objetivo es analizar el volcado de memoria `memory.raw` usando la herramienta Volatility (vol.py) para listar los procesos activos y encontrar la clave de descifrado oculta en el portapapeles.

## Detalles
- **Tipo de Artefacto**: memory_dump
- **Archivo**: memory.raw
- **Herramientas**: Volatility, Volatility 3
- **Dificultad**: intermedia
- **Tags**: memory, forensic, windows, process
