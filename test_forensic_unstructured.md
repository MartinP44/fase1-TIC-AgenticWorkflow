# Reto: Tráfico Sospechoso en el Servidor de BD

**Autor**: HackerPro (diseñado por HackerPro)
**Dificultad**: intermedia

## Descripción
Hemos capturado el tráfico de red de un servidor que fue comprometido recientemente. Creemos que el atacante extrajo datos a través de peticiones DNS inusuales. Tu tarea es analizar el archivo de tráfico `network_dump.pcap` para reconstruir la exfiltración y encontrar la bandera oculta.

Para resolver este reto necesitarás usar la herramienta Wireshark o tshark para filtrar las consultas DNS de tipo TXT.

## Detalles
- **Archivo**: network_dump.pcap
- **Categoría**: red, forense, dfir
